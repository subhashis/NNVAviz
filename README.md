# NNVAViz
Using the tool:
Dependencies: 
- flask
- keras 

```
cd ./backendSrc

# windows
$env:FLASK_APP = "app"
# Linux
export FLASK_APP=app

flask run
```

To edit the visuliaztion code you need to have Node.js installed.

Install dependencies using npm:

```
npm install
```

Compile the front-end

```
npm run build
```
